🎥 Real-time Video

Share webcam feed as beautiful ASCII art in real-time with WebRTC technology

🖥️ Terminal Native

Works in any terminal, SSH session, or console. No GUI required!

⚡ Ultra Low Latency

Direct peer-to-peer connections with ~100-200ms latency

🌍 Cross Platform

Linux, macOS, Windows support with one-line installation

🎮 Usage

Start a video session:

# Caller (video sender)
snapshell -signaled-o --room demo123 --server https://snapshell.onrender.com

# Answerer (video receiver)
snapshell -signaled-a --room demo123 --server https://snapshell.onrender.com
